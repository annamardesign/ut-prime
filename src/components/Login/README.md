# Button

## How to use

```jsx
import Login from 'ut-front-devextreme/core/Login';

<Login onClick={handlerOnClick}>Hello Login</Login>;
```

## Props

- **className** - class applied to root `Login` html element.

| propName  | propType | defaultValue | isRequired |
| --------- | -------- | ------------ | ---------- |
| className | string   |              | no         |
