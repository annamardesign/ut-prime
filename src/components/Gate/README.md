# Button

## How to use

```jsx
import Gate from 'ut-front-devextreme/core/Gate';

<Gate onClick={handlerOnClick}>Hello Gate</Gate>;
```

## Props

- **className** - class applied to root `Gate` html element.

| propName  | propType | defaultValue | isRequired |
| --------- | -------- | ------------ | ---------- |
| className | string   |              | no         |
