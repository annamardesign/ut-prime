# Navigator

## How to use

```jsx
import Navigator from 'ut-front-devextreme/core/Navigator';

<Navigator className='name' />;
```

## Props

- **className** - class applied to root `Navigator` html element.

| propName  | propType | defaultValue | isRequired |
| --------- | -------- | ------------ | ---------- |
| className | string   |              | no         |
