# Button

## How to use

```jsx
import Text from 'ut-front-devextreme/core/Text';

<Text onClick={handlerOnClick}>Hello Text</Text>;
```

## Props

- **className** - class applied to root `Text` html element.

| propName  | propType | defaultValue | isRequired |
| --------- | -------- | ------------ | ---------- |
| className | string   |              | no         |
