# Button

## How to use

```jsx
import Error from 'ut-front-devextreme/core/Error';

<Error>Error message</Error>;
```

## Props

- **className** - class applied to root `Error` html element.

| propName  | propType | defaultValue | isRequired |
| --------- | -------- | ------------ | ---------- |
| className | string   |              | no         |
