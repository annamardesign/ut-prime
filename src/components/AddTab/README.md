# Button

## How to use

```jsx
import AddTab from 'ut-front-devextreme/core/AddTab';

<AddTab onClick={handlerOnClick}>Hello AddTab</AddTab>;
```

## Props

- **className** - class applied to root `AddTab` html element.

| propName  | propType | defaultValue | isRequired |
| --------- | -------- | ------------ | ---------- |
| className | string   |              | no         |
