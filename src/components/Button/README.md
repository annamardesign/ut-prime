# Button

## How to use

```jsx
import Button from 'ut-front-devextreme/core/Button';

<Button onClick={handlerOnClick}>Hello Button</Button>;
```

## Props

- **className** - class applied to root `Button` html element.

| propName  | propType | defaultValue | isRequired |
| --------- | -------- | ------------ | ---------- |
| className | string   |              | no         |
