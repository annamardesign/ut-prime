# Button

## How to use

```jsx
import Main from 'ut-front-devextreme/core/Main';

<Main onClick={handlerOnClick}>Hello Main</Main>;
```

## Props

- **className** - class applied to root `Main` html element.

| propName  | propType | defaultValue | isRequired |
| --------- | -------- | ------------ | ---------- |
| className | string   |              | no         |
