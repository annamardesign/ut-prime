# Report

## How to use

```jsx
import Report from 'ut-front-devextreme/core/Report';

<Report className='name' />;
```

## Props

- **className** - class applied to root `Report` html element.

| propName  | propType | defaultValue | isRequired |
| --------- | -------- | ------------ | ---------- |
| className | string   |              | no         |
