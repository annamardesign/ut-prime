# Button

## How to use

```jsx
import Portal from 'ut-front-devextreme/core/Portal';

<Portal onClick={handlerOnClick}>Hello Portal</Portal>;
```

## Props

- **className** - class applied to root `Portal` html element.

| propName  | propType | defaultValue | isRequired |
| --------- | -------- | ------------ | ---------- |
| className | string   |              | no         |
