# Explorer

## How to use

```jsx
import Explorer from 'ut-front-devextreme/core/Explorer';

<Explorer className='name' />;
```

## Props

- **className** - class applied to root `Explorer` html element.

| propName  | propType | defaultValue | isRequired |
| --------- | -------- | ------------ | ---------- |
| className | string   |              | no         |
