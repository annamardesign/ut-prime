# Button

## How to use

```jsx
import App from 'ut-front-devextreme/core/App';

<App onClick={handlerOnClick}>Hello App</App>;
```

## Props

- **className** - class applied to root `App` html element.

| propName  | propType | defaultValue | isRequired |
| --------- | -------- | ------------ | ---------- |
| className | string   |              | no         |
