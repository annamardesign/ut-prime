# Button

## How to use

```jsx
import Pages from 'ut-front-devextreme/core/Pages';

<Pages onClick={handlerOnClick}>Hello Pages</Pages>;
```

## Props

- **className** - class applied to root `Pages` html element.

| propName  | propType | defaultValue | isRequired |
| --------- | -------- | ------------ | ---------- |
| className | string   |              | no         |
