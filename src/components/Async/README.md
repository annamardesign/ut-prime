# Button

## How to use

```jsx
import Async from 'ut-front-devextreme/core/Async';

<Async onClick={handlerOnClick}>Hello Async</Async>;
```

## Props

- **className** - class applied to root `Async` html element.

| propName  | propType | defaultValue | isRequired |
| --------- | -------- | ------------ | ---------- |
| className | string   |              | no         |
