# Button

## How to use

```jsx
import Loader from 'ut-front-devextreme/core/Loader';

<Loader onClick={handlerOnClick}>Hello Loader</Loader>;
```

## Props

- **className** - class applied to root `Loader` html element.

| propName  | propType | defaultValue | isRequired |
| --------- | -------- | ------------ | ---------- |
| className | string   |              | no         |
