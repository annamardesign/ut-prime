# ThumbIndex

## How to use

```jsx
import ThumbIndex from 'ut-prime/core/ThumbIndex';

<ThumbIndex className='name' />;
```

## Props

- **className** - class applied to root `ThumbIndex` html element.

| propName  | propType | defaultValue | isRequired |
| --------- | -------- | ------------ | ---------- |
| className | string   |              | no         |
